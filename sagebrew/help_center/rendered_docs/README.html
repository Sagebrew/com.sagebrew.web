Browser compatibility

Croppic works in most major browsers (yes, now even in IE9):

  • IE8: Untested
  • IE9: Works!
  • IE10+: Works!
  • Safari 4+: Works!
  • Firefox 3+: Works!
  • Chrome 14+: Works!
  • Opera 15+: Works!
