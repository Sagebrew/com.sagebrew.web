Install-OpenCV

shell scripts to install different version of OpenCV in different distributions of Linux

Ubuntu

if your system is Ubuntu, run the commands below.

$ cd Ubuntu
$ chmod +x * 
$ ./opencv_latest.sh

RedHat

if your system is RedHat, run the commands below.

$ cd RedHat
$ chmod +x * 
$ ./opencv_latest.sh

ArchLinux

if your system is RedHat, run the commands below.

$ cd ArchLinux
$ chmod +x * 
$ ./opencv2_4_0.sh
